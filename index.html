Edit your AngularJS forms inline ala LinkedIn profiles

To transform your existing <input> and <textarea> elements into editable elements simply wrap them with the <sk-editme model="value"></sk-editme> directive. Make sure to set the model attribute to the same value as the input/textarea element's ng-model attribute.

Demo

  • Hover over text items to see "editable" hint.
  • Click on text to enter edit mode.
  • Once editing is complete key ENTER/TAB or click outside element.
